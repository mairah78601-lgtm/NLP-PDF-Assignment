# NLP PDF Text Processing Assignment

## Project Description
This project performs NLP on a large PDF file. It extracts text and applies preprocessing, feature extraction, and visualization techniques.

## Dataset
sample.pdf (100+ pages)

## Steps Performed
- PDF text extraction using PyPDF2
- Regex-based preprocessing (remove numbers, symbols, spaces)
- Tokenization of text
- Stop word removal
- Stemming
- Lemmatization
- One Hot Encoding
- TF-IDF feature extraction
- Plotly scatter plot visualization

## Screenshots
All output screenshots are included in the `screenshots` folder.

## How to Run
1. Install required libraries:
   pip install PyPDF2 nltk scikit-learn pandas plotly

2. Open assignment.ipynb

3. Run all cells step by step

## Tools Used
Python, NLTK, Scikit-learn, Pandas, Plotly
